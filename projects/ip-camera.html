IP Camera System
Go, Postgres, FFmpeg, React

System Architecture

           
Browser ─→ Backend ─┬─→ Postgres
                    │
                    ├─→ Redis
                    │
                    ├─→ Camera Manager (Go) 
                    │   └─ Stream Worker (per camera)
                    │
                    ├─→ Detection Manager (Python) 
                    │   └─ Detection Worker (per camera)
                    │
                    ├─→ Recording Manager (Go) 
                    │   └─ Record Worker 
                    │
                    ╰─→ Cleanup Worker (global)
