Scene Viewer
Vulkan, GLFW, GLSL

Built as a part of CMU's 15-472 Real-Time Graphics course and controlled via the command line.

1.

Set up Vulkan boilerplate.

2.

Implemented rendering pipeline for .s72 scenes, CPU-side frustrum culling, and simple animation player; exposure and tonemapping control via CLI flags.

3.

Handled normal maps and Lambertian, Mirror, Environment, and PBR materials and implemented environment lighting.

4.

Created utility program using compute shaders to precompute mips for environment lighting on Lambertian and PBR materials via direct integration and GGX split-sum approximatation (Karis 2012), respectively.

5.

Handled directional, sphere, and spot lights using the representative point method (Karis 2012), and implemented per-frame shadow maps for spot lights.

6.

Implemented and optimized GPU procedural terrain generation with Marching Cubes via two-pass method (counting then computing vertices); grouped voxels into chunks yielding more efficient CPU-side culling.

7.

Achieved ~1000x more voxels (before exhausting 6GB VRAM) over naively allocating worst-case memory and a factor of N^3 fewer frustrum culling tests.
